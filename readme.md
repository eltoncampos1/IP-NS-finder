# CLI APP to find Ips and Server names over the internet

## Requirements

 - [Golang](https://go.dev/) 

## Usage
 - Download the Repo
 - Enter the project
 - Generate the build 
  ```sh
    go build
  ```

 - To find the ips
  ```sh
    ./command-line ip --host HOST
  ``` 
 - To find the Server names
  ```sh
    ./command-line server --host HOST
  ```
